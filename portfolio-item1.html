    Odyssey App

    Mobile Android Application

    Odyssey is designed to be an all-in-one vacation planning android mobile application. Users are able to create an account and log in to be able to create schedules for vacations or trips. Users will be able to get guidance on places to visit, routes to get to their destinations, help with expense tracking, and help with tracking activites they want to do.

    Tools and APIs Used:

    • Android Studios
    • Firebase
    • React Native
    • Visual Studios
    • Google Maps API
    • Yelp Fusion
